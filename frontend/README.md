# Frontend – Sistema de Gestión de Citas Médicas

Este proyecto corresponde al frontend del Sistema de Gestión de Citas Médicas desarrollado para un consultorio de traumatología.

La aplicación fue desarrollada utilizando React y se comunica con el backend a través de una API REST.


## Requisitos

- Node.js (versión 18 o superior recomendada)
- npm


## Instalación

Desde la carpeta del proyecto frontend, ejecutar:

npm install

Este comando descarga todas las dependencias necesarias y genera automáticamente la carpeta node_modules.


## Ejecución

Para iniciar la aplicación en modo desarrollo, ejecutar:

npm start

La aplicación se ejecutará en el navegador en la dirección:

http://localhost:3000


## Configuración

La URL del backend se encuentra centralizada en el archivo config.js.
En caso de ser necesario, puede modificarse para apuntar a otro entorno.

## Observaciones

La carpeta node_modules no se incluye en la entrega del proyecto, ya que se genera automáticamente mediante el comando npm install.
